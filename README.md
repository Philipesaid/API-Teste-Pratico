# Rest Product API

> API RESTfull usando Node.js, Restify, Mongoose e JWT. 


## API Endpoints

#### Products Rotas

- GET /products
- GET /products/:id
- POST /products
- PUT /products/:id
- DELETE /products/:id

#### User Rotas

- GET /user/:id
- PUT /user/:id
- DELETE /user/:id
- POST /register
- POST /auth
